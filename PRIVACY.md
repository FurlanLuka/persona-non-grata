# Privacy Policy — Persona Non Grata

**Last updated:** March 25, 2026

## What data does this extension collect?

This extension stores only:
- A list of GitHub usernames you choose to block (entered manually by you)
- Your extension preferences (filtering enabled, hide noise, show stats toggles)

This data is stored locally in your browser via `chrome.storage.sync` and syncs across your signed-in Chrome browsers. **No data is sent to any external server.**

## What data does this extension access?

The extension reads the HTML content of GitHub pull request pages to identify comment authors. It uses this information solely to hide comments from your blocked users list. It does not collect, store, or transmit any page content.

## Third-party sharing

This extension does not share any data with third parties.
